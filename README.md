# KarmaJobsCordova

This is a mobile cordova app.
This app communicates with a small api, and uses some components of the phone.

# Prerequisites

- Node >= 7
  - npm

# Install

```
# install global cli utils
npm i -g webpack cordova

# install dependencies
npm i

# Install for android
cordova platform add android

# build assets
npm run build

# run the cordova app
npm run start
```
